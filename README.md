# ByteWealth - A Modern Blog Template

ByteWealth is a fast, modern, and SEO-enhanced static blog template for the digital strategy and web design niche. It's built with React, TypeScript, and Tailwind CSS, focusing on performance, mobile-friendliness, and a clean, professional aesthetic.

## Features

- **Fully Responsive:** Mobile-first design that looks great on all devices.
- **Static & Fast:** Built as a static site for excellent performance and SEO.
- **Component-Based:** Developed with React and TypeScript for a modern, maintainable codebase.
- **Client-Side Routing:** A fast single-page application (SPA) experience.
- **SEO Enhanced:** Includes meta tags, Open Graph tags, and per-post JSON-LD structured data.
- **Content-Driven:**
  - Categorized blog page with tab-based filtering.
  - Pagination for article lists.
  - Dynamic read-time calculation.
  - Rich content formatting options.
- **Monetization Ready:** Includes placeholders for Google AdSense and a Web Monetization meta tag.
- **Comprehensive UI:** Includes About, Contact, and Legal (Disclaimer, Privacy, Terms) pages.

## Tech Stack

- **Frontend:** React 18, TypeScript
- **Styling:** Tailwind CSS (via CDN)
- **Routing:** Custom client-side routing logic
- **Deployment:** Vercel (or any static site host)

## Getting Started

This project is a static site and does not require a complex build process.

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```

2.  **Serve the files:**
    You can use any simple local web server to run the site. One of the easiest ways is to use Python's built-in server or the `serve` package from npm.

    **Using Python 3:**
    ```bash
    python3 -m http.server
    ```

    **Using `serve` (if you have Node.js):**
    ```bash
    npx serve .
    ```

3.  **Open in browser:**
    Navigate to `http://localhost:8000` (or the port specified by your server) in your web browser.
